EDA(Exploratory data analysis):
EDA (Exploratory Data Analysis) is the process of examining, summarizing, and visualizing data to understand its main characteristics before applying any statistical model or machine learning algorithm.
It is an important step in Data Analysis, Data Science, and Machine Learning because it helps to:
Understand the structure of the dataset
Find patterns and relationships
Detect errors or missing values
Identify outliers
Decide which methods to use for further analysis

#STEPS:
-> Data collection 
-> Data understanding
-> Data cleaning
-> Data transformation
-> Data visualization
